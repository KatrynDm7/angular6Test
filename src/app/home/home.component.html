The project is developed on the basis of angular 6 dev-starter kit.

For such components and functionality as table, pagination, cards, sorting - were used ng-prime components.

But Filter component is fully self-written.

Bonuses: localization (RU/EN).

Data is generated into-json file with 400 users: https://mockaroo.com/

Fields: id, first_name, last_name, email, department, phone
